## Data Structure: Heap

For this task, we chose to study Heap.
A heap is a special type of binary tree.
In a max heap, the parent node is always larger than its children.
Heap can be used in situations like priority queues or when we need to quickly get the largest value.

## Algorithm: Heap Sort

We also learned Heap Sort.
Heap sort first builds a max heap, and then it keeps moving the largest element to the end of the array.

Time Complexity:
- Best case: O(n log n)
- Average case: O(n log n)
- Worst case: O(n log n)
Heap sort is relatively efficient and does not need extra space.
